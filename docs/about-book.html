周深采访Talking合集

关于本书

只要你们想我的时候就可以打开那个文档。——周深

1. 整理本书初衷

  采访中的深深，既有趣活泼，又催人积极上进，说过的话，值得被文字记录。

2. 本书结构

  本书分三部分，第一部分前言及关于本书等内容，第二部分为书的主题部分，按照时间顺序，整理周深2014年出道以为的各类采访的文字稿，第三部分为附录。

3. 阅读在线版提示

  本书在线版本地址为https://CharlieTalking.com。在线版可获得更好的阅读体验——对talking重要部分进行着色区分；添加链接引用，可直接跳转相关采访视频地址；可以发表支持markdown格式的评论；支持关键字全文搜索。

4. 如何为本书做贡献

  本书主体部分按照时间顺序进行整理，但考虑工作量较大，欢迎提供整理好的文字采访。关于本书，如果您有任何意见或建议，欢迎在页面底下评论区评论（评论需要注册github账号，可直接点击评论框下方按钮进行注册，如有问题，点击查看注册教程，github为最大代码托管平台，点击查看github百度百科），也可移步小红书搜索“shenshen_929”私信我。
